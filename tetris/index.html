Canvas Tetris

A matrix-stacking retro puzzle engineered with multi-dimensional array rotations.

About the Game

Manipulate falling geometric Tetrominoes into solid horizontal rows. Completing an entire line wipes the block row from the matrix grid, awarding score tiers and dropping down all remaining structures above. The game terminates instantly if new shapes are blocked from spawning at the ceiling or if locked block stacks cross the upper boundary edge.

Controls

Use Arrow Left / Right or A / D to shift pieces horizontally.
Press Arrow Up or W to rotate shapes clockwise.
Hold Arrow Down or S to trigger a fast soft drop gravity acceleration.
