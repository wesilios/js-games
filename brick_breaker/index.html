Brick Breaker

An arcade destruction classic featuring matrix arrays and dynamic bounding-box reflections.

About the Game

Clear rows of colorful bricks by deflecting a high-velocity ball using a player-controlled paddle platform. Each brick target is tracked through an interactive 2D matrix array. When a bounding box collision is registered, the vector reflections flip automatically and the brick entity flags are set to inactive.

Controls

Slide your Mouse Left and Right across the canvas container to steer the paddle platform. Click the screen to launch the ball from the paddle and start the match!
