Canvas Shmup

An entity-component array vector shooter engineered with high-frequency bounding frame checks.

About the Game

Defend the outer orbital grid from a descending wave-based invader grid fleet. Master positioning kinematics to weave between falling enemy plasma beams while managing active laser projectile arrays. Neutralizing high-tier ships grants accelerated point structures, while allowing hostiles to pass the lower baseline causes a game termination event.

Controls

Use Arrow Left / Right or A / D to steer your spacecraft horizontally.
Press Spacebar to fire your point defense primary weapon.
