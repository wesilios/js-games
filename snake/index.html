Canvas Snake

A grid-locked retro arcade game engineered with discrete interval arrays.

About the Game

Guide the snake through a 2D matrix grid to collect spawning food units. Every piece consumed increases your score and appends a new segment to the trailing tail array. The game ends immediately if the snake collides with the outer boundary walls or crashes into its own body matrix.

Controls

Use the Arrow Keys or WASD on your keyboard to change direction. Press any movement key on the start screen to begin!
